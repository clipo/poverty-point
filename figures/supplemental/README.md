## Note on Figures S3 and S4

The source scripts for `figure_S03_conceptual_signaling_system.{png,pdf}` and `figure_S04_network_feedback_loop.{png,pdf}` are not currently identified in the codebase. The figures themselves are preserved as static assets in this folder. Reconstructing the source scripts (or replacing them with hand-drawn figures) is identified as a follow-up housekeeping task.

## Regeneration

To regenerate the renumbered/named figures, run `make figures` from the project root. Scripts write directly to this folder using these filenames; no manual copy step is required.
