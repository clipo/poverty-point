# Reference Index

Master lookup table for all sources used in the Poverty Point Costly Signaling project. Search by tags or keywords to find the relevant paper, then check for a summary or claims extraction.

## Format

```
filename | authors_year | tags | key_content
```

## Tag Vocabulary

- `critique`: Papers critiquing or responding to other claims
- `methodology`: Papers primarily about methods
- `review`: Review articles or meta-analyses
- `empirical`: Papers presenting primary data
- `modeling`: Computational or statistical modeling papers
- `ethnohistoric`: Ethnohistoric or ethnographic accounts
- `chronology`: Dating evidence
- `site-description`: Site descriptions, maps, excavation reports
- `exchange`: Long-distance exchange, exotic materials, trade networks
- `subsistence`: Subsistence strategies, resource use, diet
- `construction`: Monument construction, earthworks, labor investment
- `settlement`: Settlement patterns, aggregation, mobility
- `environment`: Paleoclimate, ecology, resource availability
- `signaling`: Costly signaling, cooperation, social organization
- `lithics`: Stone tools, PPOs, microblades
