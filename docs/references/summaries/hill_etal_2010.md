## Relevance to Argument

This study provides the first systematic geochemical sourcing of Poverty Point copper artifacts, directly bearing on the costly signaling model in two ways. First, it confirms that copper at the site is non-local and required long-distance acquisition, establishing the acquisition cost necessary for copper to function as a costly signal. Second, and more provocatively, it challenges the long-standing assumption that all Poverty Point copper originated in the Lake Superior region (~1,600 km), finding instead that over half the specimens match Canadian Maritime sources (Nova Scotia, ~2,500+ km) more closely. If confirmed, this would imply even higher acquisition costs than previously assumed. However, the small sample size (n=6), the absence of southern Appalachian copper sources from the comparative database, and the authors' own caution about the preliminary nature of results require that these findings be treated as suggestive rather than conclusive. For the model, the key takeaway is that the exchange network was likely more complex than a single Lake Superior connection, potentially involving multiple source regions and pathways.

## Data Presented

The study presents LA-ICP-MS trace element data for six copper artifacts from Poverty Point, with four sub-samples collected from each artifact and a minimum of two sub-samples from each of 61 geological source specimens. The source specimens represent six broad regions: Cap d'Or, Nova Scotia (n=3); Trout River, Newfoundland (n=4); Centennial Mine, Michigan (n=6); Champion Mine, Michigan (n=10); Minesota Mine, Michigan (n=5); Michipicoten Island, Ontario (n=6); Minong Mine, Michigan (n=8); Snake River, Minnesota (n=10); Cornwall, Pennsylvania (n=1); Greenstone, Pennsylvania (n=2); Adams County, Pennsylvania (n=5); Gold Hill, North Carolina (n=1). Compositional data for 30 trace elements were analyzed via discriminant function analysis. Results include group classification probabilities for each artifact sub-sample. Table 1 provides artifact descriptions and weights; Table 2 lists source specimen counts by region; Figures 3-5 show discriminant function plots.

## Methodological Notes

This is an unpublished symposium paper marked "DO NOT CITE WITHOUT PERMISSION OF THE AUTHORS," which limits its citability. The analytical method (LA-ICP-MS) is well-established for copper sourcing. The primary limitation is the comparative database: 61 source specimens from six regions is a reasonable but incomplete reference set, and the absence of southern Appalachian copper sources (Georgia/Alabama) is a significant gap given that steatite from that region is documented at Poverty Point. The Nova Scotia assignments could potentially shift if Appalachian sources were included and proved compositionally similar. The small artifact sample (n=6) means these results cannot be generalized to all Poverty Point copper. The internal heterogeneity of copper (requiring multiple sub-samples per specimen) complicates source assignment, as some specimens show affinities to multiple source regions.
